# planets-api
Star Wars Planets Golang API


## Requirements:
    
- [Go](https://golang.org/) (v1.10.4)
- [Glide](http://glide.sh/) (v0.13.1-3)

## Instalation:
    
    glide install


## Extras:
### Add More Dependencies:
    glide get github.com/foo/bar
or
    
    glide get github.com/foo/bar#^1.2.3
