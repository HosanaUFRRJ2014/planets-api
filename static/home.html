Welcome to Star Wars Planets API!

What you can do:

List all registered planets

GET /planets/api/planets

Create a planet

POST /planets/api/create

Example:

{ "name": "Tatooine", "terrain": "dessert", "climate": "arid" }

Search a planet by name

GET /planets/api/search?name={NAME}

Search a planet by id

GET /planets/api/search?id={ID}

Remove a planet by name

DELETE /planets/api/delete?name={NAME}

Remove a planet by id

DELETE /planets/api/delete?id={ID}
